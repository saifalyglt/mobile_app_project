# Changelog

All notable changes to the Hostel Finder app will be documented in this file.

## [1.0.0] - 2024-11-15

### Added
- Initial release of Hostel Finder app
- Complete onboarding flow with 3 slides
- Authentication system with email/password sign in/sign up
- Multi-role support (Student, Owner, Admin)
- Home screen with search and featured hostels
- Hostel listing screen with filters and sorting
- Hostel detail screen with gallery, amenities, reviews
- Booking flow with room selection, dates, and mock payment
- User profile with bookings history and favorites
- Owner dashboard for managing hostels and bookings
- Admin dashboard for verifying owners and viewing metrics
- Reviews and ratings system
- Mock backend service for demo functionality
- Firebase integration stubs for future implementation
- Localization support (English and Urdu placeholders)
- Unit tests and widget tests
- Comprehensive README with setup instructions

### Features
- Browse hostels by city with search functionality
- Filter hostels by price, amenities, and location
- Sort hostels by price, rating, or newest
- View detailed hostel information with images
- Book rooms with date selection
- Leave reviews and ratings
- Save favorite hostels
- Owner registration and verification flow
- Admin tools for managing the platform

### Technical
- Flutter 3.x / 4.x compatible
- Riverpod for state management
- go_router for navigation
- Material Design 3 theme
- Mock backend service with sample data
- Offline caching support
- Responsive UI for common phone sizes

### Documentation
- Comprehensive README with setup instructions
- API documentation for mock backend
- Firebase setup guide
- Testing documentation
- Code structure documentation
